## ✨ Overview

**Moon Calendar** is a visually rich, data-accurate moon phase explorer that combines:

- 🌌 Real-time 3D rendering  
- 🌙 Astronomical calculations  
- 📊 Data visualization  
- ⚡ Smooth UI/UX animations  

Designed as a **portfolio-grade + production-ready application**, it demonstrates advanced frontend engineering and interactive graphics.

## 🔥 Key Features

### 🌙 3D Moon Experience
- Interactive **NASA-textured 3D Moon**
- Drag to rotate, scroll to zoom
- Keyboard-accessible navigation
- Realistic lighting based on lunar phase

---

### 📅 Smart Moon Calendar
- Monthly moon phase grid
- Mini canvas-based moon rendering
- Full moon & new moon highlighting
- Smooth month navigation + quick jump controls

---

### 🧠 Accurate Astronomy Engine
- High-precision **ephemeris-based calculations**
- Fallback fast approximation mode
- Real-time:
  - Illumination %
  - Moon age
  - Cycle phase
  - Distance from Earth

---

### 📊 Data Visualization
- Monthly illumination trends (Recharts)
- Phase progression tracking
- Clean, responsive charts

---

### ⚡ Premium UI/UX
- Framer Motion animations
- Reduced-motion accessibility support
- Fully responsive (mobile + desktop)
- Keyboard-friendly interactions

---

### ♿ Accessibility
- Focus-visible states
- Semantic structure
- Motion preference handling

---

## 🎮 Keyboard Controls

| Key | Action |
|-----|--------|
| ←   | Previous day |
| →   | Next day |
| T   | Jump to today |
| + / - | Zoom 3D moon |
| Arrow keys (focused) | Rotate moon |
| Home | Reset view |
